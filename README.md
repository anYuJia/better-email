## ⚡ 什么是 Better Email？

Better Email 是一款回归本质的桌面邮箱客户端。拒绝臃肿的 Web 套壳与信息流干扰，我们以 **本地优先** 与 **极简美学** 重塑了日常邮件处理工作流。

---

## ✨ 核心优势

| 🔒 绝对隐私 | 🚀 极致性能 | 🎨 极简秩序 |
| :--- | :--- | :--- |
| **本地优先存储**<br>邮件、联系人均保存在本地 SQLite 中，不上传任何云端。 | **Rust 后端引擎**<br>基于 Rust 与系统原生 WebView 构建，内存占用极低。 | **经典三栏布局**<br>文件夹、列表、阅读区一目了然，动作触手可及。 |
| **系统级凭据安全**<br>账号密码与 Token 严格托管于系统 Keychain。 | **按需轻量加载**<br>写信、设置惰性加载；附件正文按需拉取。 | **无干扰工作台**<br>内置防追踪器，默认阻止一切远程追踪图片。 |

---

## 🛠️ 极速上手

确保您的设备已配置 [Node.js](https://nodejs.org/) 与 [Rust](https://www.rust-lang.org/) 环境。

```bash
# 1. 安装依赖
npm install

# 2. 启动本地开发环境
npm run tauri dev
```
